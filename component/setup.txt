Product: WSO2 IS authenticator for Semantic VIP
Pre-requisites:

- Maven 3.x
- Java 1.7 or above

Tested Platform: 

- Mac OSx 10.9
- WSO2 IS 5.1.0
- Java 1.7

Do the following:

Deploying and Configuring Semantic VIP artifacts:
1.  Build the <SEMANTICVIP_AUTHENTICATOR_HOME>/component/authentication-endpoint and copy the semanticvipauthenticationendpoint.war to <IS-HOME>/repository/deployment/server/webapps/

2.  Build the <SEMANTICVIP_AUTHENTICATOR_HOME>/component/authenticator & copy the org.wso2.carbon.extension.identity.authenticator.semanticvip.connector-1.0.0.jar
    to <IS-HOME>/repository/components/dropins

3.  Follow the steps in https://docs.wso2.com/display/ISCONNECTORS/Configuring+SemanticVIP+Authenticator
